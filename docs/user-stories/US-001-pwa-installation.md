# US-001 — Installation PWA

## User Story

En tant que joueur, je veux ajouter PADEL SCORE à l’écran d’accueil de mon téléphone afin de pouvoir le relancer comme une application avant un match.

## Valeur attendue

Réduire le temps entre la découverte du produit et son utilisation sur le terrain, sans passer par une boutique d’applications.

## Critères d’acceptation

- Android Chrome reconnaît PADEL SCORE comme une application installable ;
- l’utilisateur peut choisir « Ajouter à l’écran d’accueil » ou « Installer l’application » ;
- l’icône PADEL SCORE apparaît sur l’écran d’accueil ;
- le nom court affiché est `PADEL SCORE` ;
- le lancement installé masque l’interface habituelle du navigateur lorsque la plateforme le permet ;
- les couleurs de lancement sont cohérentes avec l’identité PADEL SCORE ;
- après une première utilisation en ligne, l’interface peut être rouverte sans réseau ;
- une limite de la reconnaissance vocale hors ligne n’est pas présentée comme un échec de l’installation.

## Validation terrain

Tester l’installation, la relance et le chargement hors ligne sur le téléphone Android utilisé lors du prochain essai.
