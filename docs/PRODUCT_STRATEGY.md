## 2. GO

### Objectif

### Objectif

- intégration logiciel de club ;
- intégration réservation ;
- mode tournoi ;
- API ;
- statistiques ;
- administration multi-terrains.

Ces éléments ne font pas partie du périmètre du MLP.
